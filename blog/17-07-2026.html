The Site is Live!

17 July 2026

Section Header

Over the past few days I've been working to get this site up and running. It's been a difficult process to navigate, testing several options for site hosting and creation. I tried the standard options like Wix or Squarespace but both were prohibitively expensive and weren't fit for purpose. I ended up opting for GitHub pages, which allows me to use HTML for the site. With the assistance of AI to guide me through the process I've got the site to a decent standard, and learned about website creation in the process.

I've used a little HTML in the past when creating a DND official style source book, but website creation is very different. There are so many factors to consider when making a site and so many creative choices to make in it's presentation especially when compared to creating a DND source book where those stylistic choices are essentially made for you. The benefit of having used that rudimentary HTML in the past is that I'm able to make modifications to the site myself without assitance, while having the basic knowledge to use informed prompts in the AI. While I'm generally against using AI for most things, I think in this case it was beneficial as it allowed me to continue my focus on learning C# in Unity, without confusing me with three new languages.

I'll be trying to update this blog once every week or two. I don't want to commit to weekly updates since it takes time to create the posts and write up what I've done. Time is valuable and I need to focus myself on upskilling for the next few months. For the next few weeks I'll be continuing with the Unity Junior Programmer Course and will also be investigating the landscaping software Gaea to see how it can be imbedded into my workflows.

I've also considered creating an accompanying YouTube channel to this blog, but I'll have to see if time allows for that. I hope anyone reading this enjoys the site, and potentially gains some insight into my processes, or general thoughts.
